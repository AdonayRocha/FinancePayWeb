# FinancePayWeb

FinancePayWeb is a modern, user-centric financial platform designed to help individuals manage their personal finances, wallet balances, cryptocurrency holdings, and user profile—all in an intuitive and responsive web interface.

## Business Model and Value

FinancePayWeb aims to provide users with:
- Visual and easy-to-use management of accounts and digital wallets
- Overview and details of cryptocurrencies and other assets
- User profile and authentication screens
- A template for open innovation in fintech projects

Built as a web application, it can be used directly by individuals wishing to organize their finances or by developers looking for a foundation to expand into educational or commercial finance tools.

## Infrastructure & Technical Model

- **Framework:** Next.js (React)
- **Language:** TypeScript
- **Styling:** Tailwind CSS and custom scoped CSS modules for each feature
- **Components:** Modular (NavBar, ActionButton, CardDisplay, etc.)
- **Pages:** Includes `/wallet`, `/criptos`, `/user` (login/profile)
- **Responsive:** Mobile-first with adaptable design for larger screens
- **Deployment:** Easily deployable on Vercel or any static host

### Key Folder Structure

- `src/app/` — App configuration (layout, pages, global styles)
- `src/components/` — Navigation, card displays, buttons, etc.
- `src/app/wallet/` — Styles and pages for wallet management
- `src/app/criptos/` — Crypto visualization and controls
- `src/app/user/` — User login/profile and styles

### Getting Started

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.

# FinancePayWeb (PT-BR)

O FinancePayWeb é uma plataforma financeira criada para ajudar usuários no gerenciamento de carteiras, ativos, criptomoedas e perfil de usuário, com visual moderno e design responsivo.

## Modelo de Negócio e Valor

O objetivo do FinancePayWeb é:
- Facilitar o gerenciamento das contas e carteiras digitais
- Apresentar visão geral e detalhada de criptomoedas e ativos financeiros
- Tela de perfil e login do usuário
- Template open source para projetos de fintech e educação financeira

A aplicação é pensada tanto para uso direto por usuários quanto para desenvolvedores que desejam expandi-la para projetos comerciais ou educacionais.

## Infraestrutura e Modelo Técnico

- **Framework:** Next.js (React)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS e módulos CSS dedicados para cada funcionalidade
- **Componentes:** Modulares (barra de navegação, botão de ação, cartão de saldo)
- **Páginas:** `/wallet`, `/criptos`, `/user`
- **Responsivo:** Mobile first e adaptável a desktops
- **Deploy:** Deploy fácil no Vercel ou hospedagem estática

### Estrutura Principal

- `src/app/` — Configuração do app (layout, páginas, estilos globais)
- `src/components/` — Navegação, cartões, botões etc.
- `src/app/wallet/` — Páginas e estilos da carteira
- `src/app/criptos/` — Visualização de criptomoedas
- `src/app/user/` — Login/perfil de usuário

### Como Executar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.
